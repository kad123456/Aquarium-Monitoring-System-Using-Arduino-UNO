# Aquarium Monitoring System Using Arduino UNO

An Arduino-based smart aquarium monitoring system that helps track essential water conditions to ensure a healthy environment for aquatic life.

## Tech Stack
- Arduino UNO
- Arduino IDE (C/C++)
- Sensors
- LCD Display (optional)

## Features
- Real-time water temperature monitoring
- Water level detection and monitoring
- Automatic alerts when parameters exceed safe limits
- Simple and user-friendly display interface
- Low-cost and efficient solution for aquarium maintenance

## How to Run
1. Clone this repository.
2. Open the Arduino code (`.ino` file) in Arduino IDE.
3. Connect all sensors and components to the Arduino UNO.
4. Upload the code to the Arduino board.
5. Power the system and monitor aquarium conditions in real time.

## Hardware
- Arduino UNO
- Temperature Sensor (DS18B20 / LM35)
- Water Level Sensor
- LCD Display (16x2) (Optional)
- Jumper Wires
- Breadboard
- USB Cable
- Power Supply

## Working
The system continuously reads data from the connected sensors and displays the current aquarium conditions. If the water temperature or level goes beyond the predefined safe range, the system generates an alert, helping users maintain a healthy environment for fish and other aquatic organisms.

## Future Enhancements
- Mobile app integration
- IoT-based remote monitoring
- Automatic water pump control
- pH and dissolved oxygen monitoring
